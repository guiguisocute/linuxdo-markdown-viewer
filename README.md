## 基本功能

导入脚本后，浏览L站主题时，头像下方会出现一个`</>`按钮

点击即可将帖子切换为源码模式或者渲染模式，转换为源码模式后，可以在顶部工具栏一键复制，也可自由使用光标选择文本复制，适配L站深色模式


![PixPin_2026-01-12_12-48-00|690x354](https://linux.do/uploads/default/original/4X/f/d/0/fd0ddeed80566fd97fc03fe87c61e3cf8a80eeda.gif)


### 图片转为外链url

勾选该选项后，复制的内容会自动把图片url转为外链形式

方便保存到本地或转发到其它站点查看
![PixPin_2026-01-12_12-07-36|690x354](https://linux.do/uploads/default/original/4X/a/b/3/ab3744416bbeeaf5cbff38589f2a9a5770d076d5.gif)


### 保留标题和用户信息

勾选该选项后，复制的内容将自动加上该话题的标题、链接、作者ID、和发布时间

如果复制的是回复，则还会加上回复者的ID，楼层号和时间戳
