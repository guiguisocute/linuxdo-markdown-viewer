L站有好多好看好玩的文本语法，有时候看到其它佬的帖子里各种花里胡哨的样式，自己却不知道怎么用，实在心痒痒，后来学习了Discourse后发现只用在url加`raw`就可以查看原帖的markdown源码，但是终究还是不太方便，遂vibe了这个脚本，希望可以帮到有同样需求的你

## 食用方法
### 基本功能
导入脚本后，浏览L站主题时，头像下方会出现一个`</>`按钮，点击即可切换为源码模式或者渲染模式，转换为源码模式后，可以在顶部工具栏一键复制，也可自由使用光标选择文本复制，适配L站深色模式

### 图片转为外链url
勾选该选项后，复制的内容会自动把图片url转为外链形式，方便保存到本地或转发到其它站点查看
### 保留标题和用户信息
勾选该选项后，复制的内容将自动加上该话题的标题、链接、作者ID、和发布时间，如果复制的是回复，则还会加上回复者的ID，楼层号和时间戳
